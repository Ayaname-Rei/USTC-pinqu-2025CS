### 新增期刊

请放入对应年月目录，建议格式：

```text

```

### 新增班级风采照片

请使用如下文件名格式：

```text
班级风采/照片名称_YYYYMMDD.jpg
```

例如：

```text
班级风采/开学典礼大合照_20250912.jpg
```
